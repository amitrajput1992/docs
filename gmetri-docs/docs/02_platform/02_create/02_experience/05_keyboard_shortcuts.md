# Keyboard Shortcuts


| Action                    | Shortcut                         |
|:--------------------------|:---------------------------------|
| Undo                      | **ctrl/cmd** + **z**             |
| Redo                      | **ctrl/cmd** + **shift** + **z** |
| Open Rules Panel          | **ctrl/cmd** + **space**         |
| Delete Element / Scene    | **ctrl/cmd** + **del**           |
| Hide Element              | **ctrl/cmd** + **h**             |
| Duplicate Element / Scene | **ctrl/cmd** + **d**             |
| Cycle Through Controls    | **ctrl/cmd** + **q**             |
| Zoom In                   | **ctrl/cmd** + **]**             |
| Zoom Out                  | **ctrl/cmd** + **[**             |
| Zoom Reset                | **ctrl/cmd** + **0**             |
